# EdenSchool MVC Template

Le fichier Models\Manager.php et Controllers\Controller.php contiennent respectivement les fonctions qui vont être automatiquement mises dans tout les Managers et Controllers.

N'oubliez pas de changer le namespace dans tous les fichiers PHP, Route.php à la ligne 30 et à la ligne 6 et votre nom / nom du project dans composer.json! :3

Et n'oubliez pas d'executer `composer install` après avoir changé le contenu de composer.json!
